## Ejecución
Para ejecutar el código debe ejecutar el comando *flutter run -d chrome --web-browser-flag "--disable-web-security"* para que el servicio de correos funcione correctamente, si esto no es necesario simplemente puede ejecutar *flutter run* e ingresar el número que corresponda a la opción de chrome. (Los comandos se deben ejecutar estando en la carpeta "campana_vacunas").
Para ingresar a un perfil puede ocupar los siguientes RUT con la contraseña "1234":

Administrador: 11.111.111-1

Secretaria: 22.222.222-2

Enfermera: 33.333.333-3

Médico: 44.444.444-4

Usuarios: 21.345.678-9, 55.555.555-5, 66.666.666-6, 77.777.777-7.

Los RUT de usuario también pueden ser usados para agendar citas desde el perfil de secretario.
